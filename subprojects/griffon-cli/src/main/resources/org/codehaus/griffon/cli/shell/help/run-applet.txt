DESCRIPTION
        run-applet

	Runs the application in applet mode.

SYNTAX
        run-applet [options]

OPTIONS
        --applet-height
                Sets the applet's height.
                (defaults to 240)
        --applet-width
                Sets the applet's width.
                (defaults to 320)
        --non-interactive
                Controls if the shell can ask for input or not.
