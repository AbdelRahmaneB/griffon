DESCRIPTION
        upgrade

	Upgrades a Griffon application from a previous version of Griffon.

SYNTAX
        upgrade [options]

OPTIONS
        --force
                Forces the upgrade even if the application is up-to-date.
        --install
                Install all matching upgrades.
        --non-interactive
                Controls if the shell can ask for input or not.
        --repository
                Name of an specific repository where the search will be performed.
