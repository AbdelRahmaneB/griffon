DESCRIPTION
        generate-view-script

	Generates a view script from an existing class.

SYNTAX
        generate-view-script [options] [name] [view] 

ARGUMENTS
        name
                The name of Matisse view class.
        view
                The name of Griffon View to be created.

OPTIONS
        --non-interactive
                Controls if the shell can ask for input or not.
        --view
                The name of Griffon View to be created.
